Selenium Login Automation Framework
📌 Project Overview

This project demonstrates an automated login testing framework built using Selenium WebDriver with Python. The framework validates authentication workflows by executing both positive and negative test scenarios.

🚀 Key Features

Automated login validation

Positive and negative test case coverage

Explicit waits (WebDriverWait) for stability

Chrome options configuration to prevent browser interruptions

Modular and reusable test structure

CI-ready project structure

🛠 Tech Stack

Python 3.11

Selenium 4

ChromeDriver

PyTest (optional integration)

Git
